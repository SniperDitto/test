함수(function)
C#, java, c++은 메서드(함수와 유사하지만 다름)
->객체 지향 언어에는 함수 대신 메서드가 있다
자바스크립트는 함수, 메서드 둘 다 존재

function : 반복되는 기능
같은 기능을 반복해서 사용한다면 함수를 만들어 반복을 없앨 수 있다.
-> 코드가 간단해짐

일종의 데이터 타입이기도 하다.

함수를 만들면 페이지를 닫기 전까지 메모리에 상주하게 된다.
-> 불러와서 실행을 시켜야 작동한다.

함수의 return
함수에서 값을 생성했다면 return을 통해 밖으로 뱉어내야 한다.
return를 썼다면 함수의 리턴값을 뱉으며 작동이 종료된다.
