Form 태그
input 태그
- 싱글 태그(닫는 태그 없음)
input의 type 속성 종류
- text : 글 입력
- password : 비밀번호(****로 표시됨), 영문&숫자만 가능
- radio : 옵션 선택 버튼(하나만 선택)
- checkbox : 여러 옵션 동시 선택 가능, 선택 해제 가능
- button : <button>태그와 기능 약간 다름(주의)
- number : 숫자 카운트, 익스플로러에서 안됨
다른 속성들
- name : 입력하는 내용이 무엇의 내용인지(id, pw 등)
- maxlength : 최대 입력 글자 수(byte 수 아님)
       (영문, 숫자는 글자당 1byte, 한글은 2byte)
- size(잘 사용x) : maxlength와 같은 역할
- placeholder : 미리 보이는 글자
- checked : 선택 옵션 사용 시 기본값
- value : 입력받는 값, 서버로 전송되는 값
       미리 설정 시 아무것도 입력되지 않았을 때 해당 값이 입력된 것으로 처리
       button에서는 버튼에 표시되는 내용
       <label>로 둘러싸면 옵션 선택 시 클릭하는 영역 조절 가능
<select>와 <option>
- 드롭다운 메뉴(옵션 선택)
